# Drift Mayhem

[Play on UnityPlay](https://play.unity.com/en/games/43552608-5f43-4315-b673-1ae2175c90ca/driftmayhem)

Drift Mayhem is a high-speed arcade vehicle shooter built around a distinctive
"lock-on and drift" mechanic.

Lock on to an enemy, throw your vehicle into a drift, and keep firing while
orbiting the target at full speed. The core of the game is choosing the right
moment and distance to initiate each drift, then using that drift to keep your
movement flowing so enemies never get a clean shot.

| Input | Function |
| --- | --- |
| W / S | Drive |
| A / D | Steering |
| E | Lock-On Toggle |

Press `E` to toggle lock-on. When locked on, the player vehicle begins drifting
and automatically fires at the targeted enemy. Hold `W` to keep accelerating
and maintain an orbit around the locked target.
